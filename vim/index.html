Introduction to Vim

October 5th, 2010

CPLUG

Nat Welch

natwelch.com / @icco

Why Vim?

  • used by lots of people
  • available on most operating systems
  • incredibly powerful

How to Install?

  • Ubuntu: sudo apt-get install vim-full
  • Fedora: sudo yum install vim
  • Mac: http://code.google.com/p/macvim/
  • Windows: http://www.vim.org/download.php#pc
  • CSC Machines: vim

How to Open?

vim filename

Modes

  • insert
  • command
  • select

Enter insert mode

o, i, a, c, s

Leave insert mode

esc, ctrl+c

IF YOU AREN'T TYPING, YOU SHOULDN'T BE IN INSERT MODE.

Movement

  • h - left
  • j - down
  • k - up
  • l - right
  • arrow keys

Command Mode

Where all of the cool shit happens

such as...

  • delete
  • copy
  • paste
  • sort
  • find and replace
  • collapse lines
  • apply indentation
  • insert many times
  • etc.

Delete

d modifier

Copy (yank)

y modifier

possible modifiers:

  • directions
  • numbers
  • characters (d, w)

Paste

{number} p

:sort

Find

/searchstring

Find and replace

:%s/find/replace/{modifier}

auto-indentation

==

10 = j

insert a lot

1000 i (type) esc

SELECT MODE!

byahhhh....

Select something

  • v - normal
  • shift v - line
  • ctrl v - block

Then use it

  • yank: y
  • delete: d
  • insert: I, A
  • indent: >
  • search + replace: :s/a/b/
